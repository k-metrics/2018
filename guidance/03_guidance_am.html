効率的で綺麗な可視化

【復習】整然データとは？

  • 整然データ（tidy data）とはデータハンドリングを習得する上で知っておくべき概念
    • 一言で表すと「（Coddの）第三正規形」
    • 整然データとは何か (Colorless Green Ideas)
    • 整然データってなに？ (Speaker Deck)
  • Rで最も使うデータ・フレーム型は整然データにしておくべき
    • 雑然データへの変換は簡単
    • 整然データなら可視化（見える化）も簡単

ggplot2とは？

  • Create Elegant Data Visualisations Using the Grammar of Graphics
    • 文字通り統一された文法で様々なグラフを描けるパッケージ
    • tidyverseシリーズなので整然データを前提とした設計
    • といっても現時点ではパイプ演算子（%>%）は使えません
  • 特徴
    • GIS（地理情報システム）などと同じレイヤー構造
    • ggplot2::aesの考え方を覚えるのがひとつのポイント
    • 標準で出来ないないことは関連パッケージで補える柔軟性
      • QQプロットはqqplotr, 格子配置はggExtra, etc

ggplot2を使ってみる

ggplot2::ggplot(data = iris,
                mapping = ggplot2::aes(x = Sepal.Width, y = Sepal.Length,
                                       colour = Species)) +
    ggplot2::geom_point()

ggplot2::ggplot(data = iris,
                mapping = ggplot2::aes(x = Species, y = Sepal.Length,
                                       colour = Species)) +
    ggplot2::geom_boxplot()

ベースを決めるggplot2::ggplot関数

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length))

ベースに上書きしていくイメージ

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length)) +
  ggplot2::geom_point()

ggplot2::aesの指定位置がポイント

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length, colour = Species)) +
  geom_point() + 
  geom_smooth(method = "lm")

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length)) +
  geom_point(aes(colour = Species)) + 
  geom_smooth(method = "lm")

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length)) +
  geom_point() + 
  geom_smooth(aes(colour = Species), method = "lm")

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length, colour = Species)) +
  geom_point() + geom_smooth(method = "lm") + 
  facet_wrap(~ Species)

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length)) +
  geom_point(aes(colour = Species)) + geom_smooth(method = "lm") +
  facet_wrap(~ Species)

ggplot(iris, aes(x = Sepal.Width, y = Sepal.Length)) +
  geom_point() + geom_smooth(aes(colour = Species), method = "lm") + 
  facet_wrap(~ Species)

主な描画関数

geom object 描画内容 類似object
geom_bar 棒グラフ geom_col
geom_boxplot 箱ひげ図 geom_violin
geom_dotplot ドットチャート
geom_histgram ヒストグラム geom_density
geom_line 折れ線グラフ geom_path
geom_smooth 回帰線
geom_text テキストラベル geom_label
